# Matching Benchmark Generators

This repository contains generators for the assignment (weighted bipartite) matching problem and maximum cardinality matching problem as a part of the [13th DIMACS Network Flows 2.0 Challenge](https://coral.ise.lehigh.edu/flow-challenge-2-0/).

Additional information and generated instances can be found at [our website](https://www.cs.rpi.edu/~slotag/dimacs.html).

Please contact [George Slota](mailto:slotag@rpi.edu) or [Anthony Fabius](mailto:fabiua@rpi.edu) if you encounter any problems building or running the code.
